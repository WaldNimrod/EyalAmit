# ארכיון מסלול המונולית (Bridge / Elementor / Woo / פרימיום)

**תאריך:** 2026-03-29  

---

## הקשר

הוחלט לנוע לכיוון **אתר WordPress רזה** בענף פיתוח נקי, תוך שימור תוכן ו-URLים קריטיים (במיוחד **עמודי QR**). מסלול האופטימיזציה הקודם (Toolset, LayerSlider, WooCommerce כבד, וכו') **אינו** מסלול ברירת המחדל לפרויקט החדש.

---

## מה נשמר במאגר

- קבצי WordPress, תיעוד, דוחות — נשארים ל**הפניה היסטורית** ולמיגרציית תוכן.
- [`ROADMAP-2026.md`](./ROADMAP-2026.md) ו-[`ACTIVE-TASK.md`](./ACTIVE-TASK.md) מתארים את המסלול הישן; **אין להרחיב** משימות פרימיום/Woo ללא אישור מפורש נגד כיוון האתר הפשוט.
