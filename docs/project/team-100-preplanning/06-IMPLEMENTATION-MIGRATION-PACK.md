## 6. תיעוד תחזוקה (טיוטה למסירה לבעלים)

1. **עדכון עמוד שירות:** ממשק עמודים → בחירת עמוד → עריכה → פרסום.
2. **אירוע חדש:** עריכת בלוק "אירוע" בדף הבית או קישור לטופס חיצוני — לפי מה שהוגדר ב-SSOT.
3. **תמונה לשיתוף:** החלפת קובץ בערכת הנושא או בשדה OG בתוסף SEO.
4. **בעיה טכנית:** יצירת קריאה לתמיכת אחסון + צירוף גרסת WP ותוספים.
