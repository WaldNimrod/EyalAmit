3. **שירותים** (דף מאגר או תפריט מורחב)
   3.1. שיעורי דיג'רידו / נגינה  
   3.2. טיפול בדיג'רידו / נשימה  
   3.3. סאונד הילינג / מדיטציית דיג'רידו  
   3.4. סדנאות  
   3.5. תיקון וחידוש כלים

3.7. **הרצאות** (מוצר — תוכן + כפתור סליקה חיצונית)
4. **אייל עמית** — אודות; מסע; מוקש דהימן (או קישור בולט לעמוד הנצחה אם נשאר נפרד).
5. **בלוג** — ארכיון + פוסטים; קטגוריות לפי [`BLOG-REVIVAL-PLAN.md`](../BLOG-REVIVAL-PLAN.md).

7. **ספרים** — מדור מרוכז (מוזג הוצאה); כרטיסים + דפי ספר ל-SEO + רכישה חיצונית.
7a. **קורסים** — פריט בתפריט ראשי; קישורים לסקולר (רב מסר).
7b. **הכשרות למטפלים** — עמוד נפרד; תוכן "יעלה בקרוב" (חלק מהמפה מיום א׳).
