## 1. עקרונות (מאושר בתכנון)

- **אתר אחד**, דומיין אחד; **הפרדה ב-UX בלבד** בין: (א) סטודיו — מרכז, (ב) הוצאה וספרים — משני, (ג) מופעים וארכיון — משני.
- **בלוג** כערוץ צמיחה ו-SEO, מקושר לשירותים.
- **עמוד EN** נפרד, ממוקד, ללא כפילות מלאה של כל האתר.

## 2. תפריט ראשי (טיוטת ניסוח — לאישור אייל)

**ענף ראשי — סטודיו (דגש ויזואלי)**

- בית (סטודיו)
- שירותים ▾  
  - לימוד / שיעורים  
  - טיפול / נשימה  
  - סאונד הילינג  
  - בניית כלי / תיקון (לפי היקף בפועל)
- בלוג
- צור קשר

**ענף משני (משני חזותית — "עוד מאייל" או שם שייבחר)**

- הוצאה וספרים  
- הופעות וארכיון  

**גלובלי**

- English (קישור לעמוד נחיתה EN)

## 3. Wireframe טקסטואלי — דף בית (סטודיו)

```
[לוגו]  [תפריט ראשי — סטודיו]     [English]

[Hero: כותרת סטודיו + משפט מפתח + CTA טלפון/צור קשר]

[שורת כרטיסים — 3–4 שירותים מרכזיים, כל אחד קישור לעמוד]

[אירוע הבא — בלוק תבנית: תאריך, שעות, משפט, קישור לטופס חיצוני]

[פס קצר: מוזה / ספרים — קישור]  [פס קצר: הופעות — קישור]

[בלוג — 3 פוסטים אחרונים + "לכל המאמרים"]

[פוטר: נגישות | תקנון | פייסבוק | טלפון]
```

## 7. עמוד נחיתה באנגלית (`/en/` או `/english/`)

**מטרה (למילוי אייל):** תיירים / קהילה בינלאומית / חיפושים באנגלית.

**מבנה מוצע:**

```
[H1 — Eyal Amit | Didgeridoo & Circular Breathing Studio, Israel]
[Sub — Pardes Hanna–Karkur area — או ניסוח מדויק]
[פסקה קצרה: מי, מה, 25+ שנות ניסיון — ללא עומס]
[3 bullets — Lessons | Healing / breath | Sound healing]
[CTA — Contact / WhatsApp / Phone]
[קישור — Full site in Hebrew]
```

**SEO / טכני**

- `lang="en"` לעמוד; `hreflang` בין `he` ל-`en` אם מיושם.
- OG נפרד לעמוד EN.
- אותם דרישות נגישות כמו לשאר האתר.
