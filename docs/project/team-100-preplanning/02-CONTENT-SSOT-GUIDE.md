## 3. עץ מידע לפי כוונת משתמש (שלד מומלץ)

התאמה לתוכן שמופיע היום ב-[אתר החי](https://www.eyalamit.co.il/) ובמאגר:

1. **מי אנחנו** — אודות אייל, הסטודיו (פרדס חנה), הקשר למסורת/מורה (מוקש דהימן).
2. **מה אנחנו מציעים** — שירותים: לימוד, טיפול, סאונד הילינג, בניית כלי, תיקון, מידע על כלים ואביזרים (**ללא** עגלה באתר החדש).
3. **למי זה מתאים** — קטעים קצרים או FAQ (תומך SEO).
4. **איך נרשמים / יוצרים קשר** — צור קשר, טלפון, קישורים לטפסים חיצוניים (למשל Google Forms).
5. **חומר עומק** — בלוג דיג'רידו (פוסטים), כתבות / עדויות — רק אם נשארים רלוונטיים למותג הסטודיו.

## 4. תוכן תאריכי (סדנאות / אירועים)

באתר החי מופיעים מודולים עם תאריכים (למשל סדנאות). להחליט במסגרת מילוי ה-SSOT:

- **עמוד קבוע** שמתעדכן ידנית, או  
- **קישור חיצוני** (טופס / לוח שנה), או  
- **בלוק "אירוע הבא"** באתר החדש ללא מערכת הזמנות מובנית.

סמן החלטה בעמוד הבית ובשורות הרלוונטיות ב-CSV.
