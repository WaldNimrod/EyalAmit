חיבור Microsoft Clarity לאתר — מדריך מהיר

5 דקות · בלי לגעת בקוד · כל מה שצריך ממך זה מזהה אחד

מה זה Clarity? כלי חינמי של מיקרוסופט שמראה לנו איך גולשים מסתובבים באתר (מפות חום והקלטות מסך) — כדי שנוכל לשפר את חוויית המשתמש ואת ההמרות.
מה צריך ממך? רק ליצור פרויקט ולשלוח לנו מזהה אחד (Project ID). את הקוד באתר אנחנו נתקין — אתה לא נוגע ב‑WordPress.
  1. היכנס לאתר של Clarity והירשם

    גלוש אל clarity.microsoft.com ולחץ Sign up (הרשמה). אפשר להתחבר עם חשבון Google, Microsoft, או אימייל — מה שנוח לך.

  2. צור פרויקט חדש

    לחץ על הכפתור + New project (פרויקט חדש).

  3. מלא את הפרטים

    • Name (שם): Eyal Amit
    • Website URL: https://www.eyalamit.co.il
    • Site category: בחר מה שמתאים (לא קריטי — למשל «Lifestyle» או «Other»).
    ואז לחץ Add new project.

  4. מצא את ה‑Project ID

    אחרי היצירה, Clarity יציג קטע קוד להתקנה (בחר «Install code manually» אם נשאל). בתוך הקוד תופיע שורה כזו — עשרת התווים המודגשים הם ה‑Project ID:

    https://www.clarity.ms/tag/xxxxxxxxxx

    טיפ: ה‑Project ID מופיע גם בכתובת הדפדפן: clarity.microsoft.com/projects/view/xxxxxxxxxx

  5. העתק ושלח לנו

    העתק את ה‑Project ID (או פשוט את כל קטע הקוד) ושלח לנו ב‑וואטסאפ או דרך ה‑Hub. זהו מבחינתך! 🎉

  6. אנחנו מסיימים את החיבור

    נטמיע את הקוד באתר ונוודא שהמדידה עובדת. אינך צריך להתקין שום דבר ב‑WordPress.

📌 מה אנחנו צריכים ממך בסוף
ה‑Project ID — רצף של כ‑10 תווים (אותיות וספרות), שמופיע אחרי clarity.ms/tag/. זה הכל.
כבר שלחת לנו את Google Analytics (GA4) — תודה! ✅   Clarity הוא הפריט האחרון להשלמת המדידה.
