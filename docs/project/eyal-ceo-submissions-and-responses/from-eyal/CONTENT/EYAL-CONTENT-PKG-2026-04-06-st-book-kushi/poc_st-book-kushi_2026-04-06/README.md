# POC - st-book-kushi

חבילה זו מדגימה את תהליך העבודה המלא לפי הקאנון המעודכן.

## מה כלול
- קובץ Markdown רשמי להגשת תוכן העמוד
- קובץ JSON רשמי להגשת תוכן העמוד
- קובץ JSON ייעודי לאינסטנסים מסוג Gallery

## מטרת ה-POC
1. להדגים שימוש ב-`tpl-book-detail`
2. להראות הפרדה בין תוכן עמוד לבין גלריה
3. להוכיח שהתהליך ישים לעמוד אמיתי מתוך עץ האתר הנעול

## הערות
- URL הרכישה הראשי נלקח מעץ האתר הנעול.
- URL הרכישה המשני נלקח מהקישור הקיים לספר האלקטרוני.
- שמות קבצי הגלריה מבוססים על הנכסים הקיימים בעמוד הנוכחי ויש להשלים אותם מול Drive לפני פריסה.
