חבילת הגשה - עמוד bundle חדש תחת מוזה הוצאה לאור
================================================

מה בחבילה
---------
1. קובץ Markdown קנוני להגשת התוכן
2. קובץ JSON קנוני להגשת התוכן
3. קובץ JSON להצעת תוספת לעץ האתר
4. קובץ Markdown מסכם להצעת התוספת

הערות
-----
- העמוד החדש אינו חלק מהעץ הנעול המקורי.
- לכן יש לאשר קודם את צומת `st-book-bundle` ואז לבצע ingest לקובצי התוכן.
- התוכן נבנה על בסיס החלטת עבודה: עמוד מבצעים חדש, אך עם מבצע אחד בלבד - bundle של שלושת הספרים.
- התמחור מבוסס על עמוד המבצעים הישן: 150 ש"ח במקום 207 ש"ח.
- URL הסליקה של החבילה טרם סופק, ולכן סומן כ-TODO.
