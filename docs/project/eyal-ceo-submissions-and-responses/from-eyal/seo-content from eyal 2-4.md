מסמך אפיון אתר סופי – אייל עמית
GEO / AEO / SEO
(גרסה מעודכנת)



1. עקרונות מבנה (למתכנת)
• האתר בנוי סביב ליבה אחת: טיפול בנשימה באמצעות דיג'רידו.
• היררכיה ברורה: טיפול → שיטה → שירותים → תוכן → מוצרים.
• כל עמוד עונה על שאלה (AEO).
• מבנה ברור למנועי חיפוש (SEO).
• יצירת סמכות (GEO) באמצעות שיטה, ניסיון ותוכן עומק.


2. היגיון מבנה (SEO / GEO / AEO)
• GEO: בניית סמכות סביב 'שיטה' ו'מרכז'.
• AEO: כל עמוד נותן תשובה ברורה לשאלה.
• SEO: הפרדה בין עולמות תוכן.
• UX: ניווט פשוט והיררכי.
