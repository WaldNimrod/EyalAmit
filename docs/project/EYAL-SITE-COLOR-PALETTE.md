## 2. מיפוי סמנטי ל־CSS (Hub ומוקאפים)

ב־`hub/src/assets/hub.css` מוגדרים משתנים גלובליים. שימוש מומלץ בכל קובץ חדש:

| משתנה | ערך |
|--------|-----|
| `--eyal-sand` … `--eyal-brick` | הצבעים הגולמיים מהטבלה לעיל |
| `--eyal-accent-rgb` | `164, 78, 43` (מ־Terracotta) לשקיפויות `rgba(...)` |
| `--accent` | Terracotta — ברירת מחדל להדגשת ממשק |
| `--accent-strong` | Brick — כפתורי ייצוא / פעולה בולטת |
| `--text` | Ink |
| `--text-muted` | Earth |

אין להחזיר צבעי סייד־בר (`#8b4513` וכו׳) כברירת מחדל בפרויקט.

---

## 3. יישום

- **Hub:** `hub/src/assets/hub.css` (נבנה ל־`hub/dist/assets/hub.css`).
- **מוקאפים עצמאיים:** להעתיק את בלוק `:root` מה־Hub או לקשר למסמך זה כמקור.
- **וורדפרס / תבנית סופית:** להגדיר אותם טוקנים ב־theme settings / CSS variables.
