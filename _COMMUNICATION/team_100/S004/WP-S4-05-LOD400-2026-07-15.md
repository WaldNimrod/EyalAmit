# WP-S4-05 — LOD400: שחזור עריכת wp-admin לעמודי Chapters (free-ACF, מונחה-נתונים)

> **צרכן:** team_110 (בנייה על Cursor Composer). **ולידציה חוצת-מנוע:** team_90 (מנוע שונה — Iron Rule #1).
> **הכרעת מודל:** אופציה B מ-[WP-S4-05-LOD300](WP-S4-05-LOD300-FIELD-MODEL-2026-07-15.md) — free-ACF, קבוצות לפי-סוג-עמוד, רישום מונחה-נתונים מתוך ה-defaults, overlay יחיד למסלול-B.
> **כלל ברזל של המערכת:** כל שדה חייב **נפילה בטוחה** — ריק/ACF-כבוי → seeded default; לעולם לא מסך לבן.

## 0. סיכום מנהלים של הבנייה (מה נבנה)
1. **`inc/chapters/acf-fields-inner.php`** — קובץ יחיד שמייצר, מתוך מערך ה-defaults של כל סוג-עמוד, קבוצת-שדות free-ACF לפי כלל שמות דטרמיניסטי (מסלול-B) + קבוצת method (flat-key).
2. **accessors + overlay** ב-`chapters-render.php`: `ea_chapters_page_sections()` שמזליג ACF לתוך מערך ה-`sections` לפי אינדקס, ו-`ea_chapters_section_specs()` המונחה-סוג.
3. **`ea_chapters_repeater_specs()` מודע-סוג** — פותר התנגשות שמות (`whom_items`/`testi_items`/`mag_items`) בין home ל-method.
4. **עריכת התבניות** `tpl-chapters-page.php` + `tpl-chapters-mokesh.php` — לצרוך את ה-overlay במקום לקרוא defaults ישירות, ולפענח תמונות דרך `ea_chapters_resolve_img()`.
5. **meta-box מחיר** — `inc/chapters/product-meta.php`: UI לעריכת `ea_product_price` (postmeta) ל-5 עמודי המוצר + שדה `price` לספרים.
6. **שורות `require`** ב-`chapters-bootstrap.php`.

**עיקרון מנחה:** אין כפילות סכימה. ה-defaults הוא ה-SSOT; הרישום וה-overlay גוזרים ממנו את שמות השדות באותו כלל דטרמיניסטי, כך ששניהם לעולם לא נפרדים.

## 2. כלל השמות הדטרמיניסטי (חוזה מפתח/שם)

### 2.1 מסלול-B — לפי אינדקס-סקשן
עבור עמוד מסוג `<type>`, סקשן באינדקס `N` (0-based) מסוג `part`, ארג `<arg>`:

```
scalar field name : s{N}_{arg}                 e.g. s3_body, s0_title
scalar field key  : f_{type}_s{N}_{arg}        e.g. f_treatment_s3_body
list slot name    : s{N}_i{K}_{sub}            e.g. s5_i2_name  (K = 1..count)
list slot key     : f_{type}_s{N}_i{K}_{sub}
list repeater name (specs registry) : {type}_s{N}
field group key   : group_chapters_{type}      e.g. group_chapters_treatment
group location    : page == <slug>             (slug מ-ea_chapters_route_map inverse)
```

### 2.2 מסלול-A (method) — flat-key
שמות השדה **זהים למפתחות ה-defaults ולשמות שה-accessors מבקשים** (`phero_title`, `split_body`, `mag_items_*`, וכו'). מפתחות: `f_method_{name}`. קבוצה: `group_chapters_method`.

### 2.3 תמונות
כל שדה תמונה: `type=image, return_format=id, preview_size=medium, library=all`. וידאו: `type=file, return_format=id, mime_types=mp4`.

## 3. הרישום המונחה-נתונים (הלב של הבנייה)

### 3.1 קובץ חדש: `site/wp-content/themes/ea-eyalamit/inc/chapters/acf-fields-inner.php`
נטען מתוך `chapters-bootstrap.php` (§6). מבנה כללי:

```php
<?php
defined( 'ABSPATH' ) || exit;
add_action( 'acf/init', 'ea_chapters_register_inner_fields' );

/**
 * Editable-arg whitelist per part type. Anything NOT listed is structural and locked.
 * 'img'  => image field; 'file' => file; 'wys' => wysiwyg; 'txt'/'ta' => text/textarea.
 * 'list' => array of items[]; each list gives sub => kind.
 */
function ea_chapters_part_field_map() {
	return array(
		'phero'        => array( 'scalars' => array( 'chap'=>'txt','title'=>'ta','sub'=>'ta','media'=>'img','media_alt'=>'txt','cta_label'=>'txt','cta_url'=>'txt' ) ),
		'prose'        => array( 'scalars' => array( 'chap'=>'txt','title'=>'txt','body'=>'wys','toggle_label'=>'txt' ) ),
		'split'        => array( 'scalars' => array( 'chap'=>'txt','title'=>'txt','body'=>'wys','image'=>'img','alt'=>'txt' ) ),
		'lead'         => array( 'scalars' => array( 'chap'=>'txt','title'=>'txt','lead'=>'ta' ) ),
		'bleed'        => array( 'scalars' => array( 'image'=>'img','alt'=>'txt','quote'=>'ta','attrib'=>'txt' ) ),
		'cta'          => array( 'scalars' => array( 'title'=>'ta','body'=>'ta','cta_label'=>'txt','cta_url'=>'txt','cta2_label'=>'txt','cta2_url'=>'txt' ) ),
		'videoblk'     => array( 'scalars' => array( 'chap'=>'txt','title'=>'txt','body'=>'wys','poster'=>'img','video'=>'file','cap'=>'txt','alt'=>'txt' ) ),
		'mag'          => array( 'scalars' => array( 'chap'=>'txt','title'=>'txt','image'=>'img','alt'=>'txt','cap_b'=>'txt','cap_sub'=>'txt' ), 'list' => array( 'title'=>'txt','text'=>'ta' ) ),
		'steps'        => array( 'scalars' => array( 'chap'=>'txt','title'=>'txt','lead'=>'ta' ), 'list' => array( 'title'=>'txt','text'=>'ta' ) ),
		'reveals'      => array( 'scalars' => array( 'chap'=>'txt','title'=>'txt','lead'=>'ta' ), 'list' => array( 'image'=>'img','title'=>'txt','more'=>'ta' ) ),
		'dd'           => array( 'scalars' => array( 'chap'=>'txt','title'=>'txt','lead'=>'ta' ), 'list' => array( 'tag'=>'txt','title'=>'txt','body'=>'wys' ) ),
		'testimonials' => array( 'scalars' => array( 'chap'=>'txt','title'=>'txt' ), 'list' => array( 'name'=>'txt','text'=>'ta' ) ),
		'timeline'     => array( 'scalars' => array( 'chap'=>'txt','title'=>'txt' ), 'list' => array( 'year'=>'txt','text'=>'ta' ) ),
		'gallery'      => array( 'scalars' => array( 'chap'=>'txt','title'=>'txt','lead'=>'ta' ), 'list' => array( 'image'=>'img','alt'=>'txt','cap'=>'txt' ) ),
		'bookcard'     => array( 'scalars' => array( 'chap'=>'txt','title'=>'txt' ), 'list' => array( 'cover'=>'img','meta'=>'txt','title'=>'txt','blurb'=>'ta','url'=>'txt' ) ),
		'product-cta'  => array( 'scalars' => array( 'title'=>'txt','body'=>'ta','contact_label'=>'txt','price_note'=>'txt' ) ), // price via meta-box, not ACF
		'faqblock'     => array( 'scalars' => array( 'chap'=>'txt','title'=>'txt' ) ), // items = CPT ea_faq (locked)
		'fbembeds'     => array( 'scalars' => array( 'chap'=>'txt','title'=>'txt' ) ), // hrefs locked
		'contact'      => array( 'scalars' => array() ),
	);
}

/** List-slot headroom: registered slots = default count + headroom (min 1). */
function ea_chapters_list_headroom( $part ) {
	return ( 'testimonials' === $part ) ? 4 : 2;
}
```

**הלוגיקה:** `ea_chapters_register_inner_fields()` עוברת על כל סוגי מסלול-B (מקור: היפוך `ea_chapters_route_map()` → slug→type, פחות type-ים נעולים: `qr-hub, qr, galleries, media, en`), לכל type טוענת `require defaults/{type}-defaults.php`, ואז:
- רושמת `phero_*` מ-`$d['phero']` לפי המפה של `phero`.
- עוברת על `$d['sections']` עם אינדקס `N`; לכל סקשן, לפי `part`, רושמת שדות סקלריים `s{N}_{arg}` **רק** לארגים שקיימים ב-map ובקטלוג; ואם ל-part יש `list`, רושמת fixed-slots `s{N}_i{K}_{sub}` עבור `K=1..(count($d..items)+headroom)` ומוסיפה רשומת specs `{type}_s{N} => {count, subs}`.
- כל קבוצה נרשמת ב-`acf_add_local_field_group` עם `location = [[ page == slug ]]`, `hide_on_screen=['the_content']`, `active=true`, כותרת עברית «פרקים — <שם עמוד>».

**מפתח:** מכיוון ש-`$default` לכל שדה זמין בזמן הרישום, אפשר גם למלא `'instructions'` עם תקציר ה-default (עזר לעורך). אין חובה.

### 3.2 חוזה ה-«נעול» (לא-שדות)
`part, figr, reversed, id, alt(bool), center, dark, collapsible, active, cats, slug, cta_slug, yt_id, pending, pending_label` — **לא** נרשמים כשדות; נשארים מה-defaults דרך ה-overlay (§4).

### 3.3 קבוצת method (flat-key) — באותו קובץ
פונקציה נפרדת `ea_chapters_register_method_fields()` (או ענף באותה acf/init) שרושמת `group_chapters_method` עם שמות שטוחים **התואמים למה שהתבנית מבקשת** (§1.1). location: `page == method`. רשימות: `mag_items/whom_items/testi_items` בשמות אלו בדיוק (כי התבנית קוראת `ea_chapters_rows('whom_items')` וכו'), עם slots לפי §4.2.

## 4. accessors + overlay + repeater-specs מודע-סוג

### 4.1 `ea_chapters_repeater_specs()` → מודע-סוג (עריכה ב-`chapters-render.php`)
**בעיה מאומתת:** `whom_items` ב-home = `{image,text}` count 4, אך ב-method = `{image,title,more}` count 4; `testi_items` ב-home = `{text,name,initial,avatar}` count 4 ואילו ב-method = `{name,text}` count 8. מכיוון ש-`ea_chapters_assemble_rows()` שולפת spec לפי שם בלבד, ספק גלובלי יחיד שובר אחד מהם.

**פתרון:** הופכים את הפונקציה למודעת-סוג פנימית (בלי לשנות חתימות שקוראות אותה):

```php
function ea_chapters_repeater_specs( $type = null ) {
	$type = ( null !== $type ) ? $type : ea_chapters_type();
	$base = array( // home (back-compat — ברירת מחדל)
		'about_timeline' => array( 'count' => 4, 'subs' => array( 'year','text' ) ),
		'whom_items'     => array( 'count' => 4, 'subs' => array( 'image','text' ) ),
		'session_cards'  => array( 'count' => 4, 'subs' => array( 'image','title','text','reveal' ) ),
		'testi_items'    => array( 'count' => 4, 'subs' => array( 'text','name','initial','avatar' ) ),
		'start_steps'    => array( 'count' => 3, 'subs' => array( 'title','text' ) ),
	);
	$by_type = array(
		'method' => array(
			'mag_items'  => array( 'count' => 6,  'subs' => array( 'title','text' ) ),
			'whom_items' => array( 'count' => 6,  'subs' => array( 'image','title','more' ) ),
			'testi_items'=> array( 'count' => 12, 'subs' => array( 'name','text' ) ),
		),
	);
	// section repeaters for path-B types are merged in from ea_chapters_section_specs($type)
	$specs = ( 'home' === $type ) ? $base : ( isset( $by_type[ $type ] ) ? $by_type[ $type ] : array() );
	return array_merge( $specs, ea_chapters_section_specs( $type ) );
}
```

- `ea_chapters_assemble_rows()` נשארת כפי שהיא (כבר קוראת `ea_chapters_repeater_specs()` ללא ארג → תופסת את הסוג הנוכחי).
- `ea_chapters_section_specs($type)` מחזירה את רשומות `{type}_s{N}` (נגזרות מאותו מיפוי כמו הרישום — פונקציית עזר משותפת כדי למנוע drift).

### 4.2 overlay למסלול-B: `ea_chapters_page_sections()` (חדש ב-`chapters-render.php`)
```php
/**
 * Overlay ACF values onto the seeded sections array, by index.
 * Scalars: ea_chapters_section_scalar(); lists: ea_chapters_section_rows().
 * Structural args pass through untouched. Image args resolved to URLs.
 * @return array[] sections ready for the foreach in the templates.
 */
function ea_chapters_page_sections() {
	$d    = ea_chapters_defaults();
	$type = ea_chapters_type();
	$out  = array();
	$secs = ( isset( $d['sections'] ) && is_array( $d['sections'] ) ) ? $d['sections'] : array();
	$map  = ea_chapters_part_field_map();
	foreach ( $secs as $N => $sec ) {
		$part = isset( $sec['part'] ) ? $sec['part'] : '';
		$args = isset( $sec['args'] ) && is_array( $sec['args'] ) ? $sec['args'] : array();
		if ( isset( $map[ $part ]['scalars'] ) ) {
			foreach ( $map[ $part ]['scalars'] as $arg => $kind ) {
				if ( ! array_key_exists( $arg, $args ) ) { continue; } // only editable args present in defaults
				$name = 's' . $N . '_' . $arg;
				if ( 'img' === $kind || 'file' === $kind ) {
					$args[ $arg ] = ea_chapters_resolve_img( ea_chapters_field( $name, null ) ?: $args[ $arg ] );
				} else {
					$args[ $arg ] = ea_chapters_field_or( $name, $args[ $arg ] ); // ACF if non-empty else default
				}
			}
		}
		if ( isset( $map[ $part ]['list'] ) ) {
			$rows = ea_chapters_rows( $type . '_s' . $N ); // assembled ACF rows or []
			if ( ! empty( $rows ) ) {
				$args['items'] = ea_chapters_merge_list_rows( $rows, isset( $args['items'] ) ? $args['items'] : array(), $map[ $part ]['list'] );
			}
		}
		$out[] = array( 'part' => $part, 'args' => $args );
	}
	return $out;
}
```

- **`ea_chapters_field_or($name,$default)`** — וריאנט חדש: מחזיר `get_field($name)` אם לא-ריק, אחרת את `$default` **שהועבר במפורש** (כי שמות ה-section אינם מפתחות top-level ב-defaults; ה-accessor הרגיל לא ידע את ה-default). פונקציה קטנה של ~6 שורות.
- **`ea_chapters_merge_list_rows()`** — ממזג rows של ACF עם ה-default, per-slot, בכלל «ריק בשדה → default של אותו slot», כדי שמילוי חלקי לא ימחק פריטי-ברירת-מחדל שלא נגעו בהם. אם ACF ריק לגמרי — נשאר ה-default כולו.
- **פענוח תמונות:** בוצע כאן דרך `ea_chapters_resolve_img()`.

### 4.3 עריכת התבניות
ב-`tpl-chapters-page.php` וב-`tpl-chapters-mokesh.php`:
- מחליפים את שליפת `$ea_sections = $ea_d['sections']` וה-`foreach` שמפענח תמונות ב:
```php
$ea_sections = ea_chapters_page_sections();
foreach ( $ea_sections as $ea_s ) {
	if ( empty( $ea_s['part'] ) ) { continue; }
	get_template_part( 'template-parts/chapters/parts/' . $ea_s['part'], null, $ea_s['args'] );

```
  (פענוח התמונות עבר ל-overlay; מסירים את לולאת `ea_chapters_asset_url` על `image/media/poster/video`.)
- **phero:** מחליפים את בניית `$ea_phero` ב-`ea_chapters_phero_overlay()` — פונקציה קטנה שמזליגה `phero_*`/`s?` על `$d['phero']` באותו דפוס (סקלרים `phero_{arg}`, תמונה דרך resolve). לחלופין, מכיוון שהתבנית קוראת `phero` כמערך אחד, אפשר להשתמש בשמות שדה `phero_{arg}` ו-overlay ייעודי; זה מיושר עם method (שכבר קורא `ea_chapters_field('phero_*')`).

> **אזהרת רגרסיה:** לוודא ש-home (מסלול-A) **לא** נוגע ב-overlay — הוא לא משתמש ב-`tpl-chapters-page.php`. method נשאר על accessors שלו; ה-overlay של מסלול-B לא רלוונטי אליו.

## 5. meta-box מחיר מוצר

### 5.1 מפתח האחסון (מאומת בקוד)
`get_post_meta( $post_id, 'ea_product_price', true )` — נקרא ב-[chapters-commerce.php](../../../site/wp-content/themes/ea-eyalamit/inc/chapters/chapters-commerce.php) (`ea_w2_05_price`), ב-[shop-defaults.php], וב-[product-cta.php]. **postmeta key = `ea_product_price` (string; מספר ש"ח ללא סימן, למשל `1450`).** כרגע **אין UI לעריכתו** → זו הסיבה שהמחיר לא ניתן לעריכה. חסר לגמרי meta-box.

### 5.2 קובץ חדש: `site/wp-content/themes/ea-eyalamit/inc/chapters/product-meta.php`
```php
<?php
defined( 'ABSPATH' ) || exit;
add_action( 'add_meta_boxes', 'ea_product_price_add_box' );
add_action( 'save_post_page', 'ea_product_price_save', 10, 2 );

function ea_product_price_is_product( $post ) {
	$slug = get_post_field( 'post_name', $post );
	return in_array( $slug, array( 'didgeridoos','bags','stands-storage','stand-floor','repair' ), true );
}
function ea_product_price_add_box() {
	add_meta_box( 'ea_product_price', 'מחיר מוצר (₪)', 'ea_product_price_render', 'page', 'side', 'high' );
}
function ea_product_price_render( $post ) {
	if ( ! ea_product_price_is_product( $post ) ) { echo '<p>שדה מחיר פעיל בעמודי מוצר בלבד.</p>'; return; }
	wp_nonce_field( 'ea_product_price_save', 'ea_product_price_nonce' );
	$v = esc_attr( (string) get_post_meta( $post->ID, 'ea_product_price', true ) );
	echo '<label for="ea_product_price_f">מחיר בש"ח (ריק = "מחיר לפי התאמה")</label>';
	echo '<input type="text" inputmode="numeric" id="ea_product_price_f" name="ea_product_price_f" value="' . $v . '" style="width:100%" />';
}
function ea_product_price_save( $post_id, $post ) {
	if ( ! isset( $_POST['ea_product_price_nonce'] ) || ! wp_verify_nonce( $_POST['ea_product_price_nonce'], 'ea_product_price_save' ) ) { return; }
	if ( defined( 'DOING_AUTOSAVE' ) && DOING_AUTOSAVE ) { return; }
	if ( ! current_user_can( 'edit_page', $post_id ) ) { return; }
	if ( ! ea_product_price_is_product( $post ) ) { return; }
	$raw = isset( $_POST['ea_product_price_f'] ) ? sanitize_text_field( wp_unslash( $_POST['ea_product_price_f'] ) ) : '';
	$raw = preg_replace( '/[^\d.]/', '', $raw ); // digits + dot only
	if ( '' === $raw ) { delete_post_meta( $post_id, 'ea_product_price' ); }
	else { update_post_meta( $post_id, 'ea_product_price', $raw ); }
}
```

### 5.3 מחיר ספרים
ספרים (`vekatavta/kushi-blantis/tsva-bekahol`) קוראים `price` ממערך ה-defaults (`'price' => 79`), לא מ-postmeta. שדה `price` (text) ייכלל בקבוצת-השדות של כל ספר (top-level, כמו phero), ויקרא דרך `ea_chapters_field('price')`. **דורש** התאמה קטנה בתבנית/חלק שמציג את מחיר הספר כדי לקרוא accessor במקום default ישיר (זהה בדפוסו ל-overlay של phero). מסומן כתת-משימה בבדיקת S4-02 (חוב ספרים) — לתאם.

## 8. קריטריוני קבלה (AC) — מדידים

### 8.1 AC כלליים (לכל עמוד חשוב)
- **AC-EDIT:** עריכת שדה בודד ב-wp-admin (למשל `s3_body` ב-treatment) ושמירה → הטקסט המעודכן מופיע ב-front render של אותו עמוד (מחליף את ה-default).
- **AC-FALLBACK:** ניקוי אותו שדה (ריק) ושמירה → העמוד חוזר להציג את ה-seeded default; **אין** ריק/שבר.
- **AC-NOACF:** השבתת תוסף ACF (או `add_filter('acf/settings/...')` mock) → כל העמודים מרנדרים מלאים מ-defaults, **אין מסך לבן**, אין `undefined function` (guards בכל הקבצים).
- **AC-IMG:** העלאת תמונה בשדה image (attachment-id) → ה-URL הנכון מוצג ב-front (פענוח `ea_chapters_resolve_img`), ולא id גולמי/קישור שבור.
- **AC-LIST:** עריכת slot ברשימה (למשל reveals slot 2 `more`) → משתקף; slot ריק → נשאר default של אותו slot; מילוי slot מעבר לספירת-ברירת-המחדל (headroom) → פריט חדש מופיע.
- **AC-LOCK:** ארגים מבניים (`figr/reversed/dark/id/cats/…`) **אינם** מופיעים כשדות ב-wp-admin ולא ניתנים לעריכה.

### 8.2 AC ספציפיים
- **method:** עריכת `mag_items` slot 3, `whom_items` (image+title+more), `testi_items` slot 9 → כולם משתקפים; **אין** התנגשות עם home (הרצת home במקביל עדיין נכונה — `whom_items`/`testi_items` שומרים על ה-shape של home).
- **מוצר (5 עמודים):** meta-box «מחיר מוצר (₪)» מופיע ב-side, שמירת מספר → `ea_product_price` נשמר ומופיע ב-product-cta וב-shop; ריק → «מחיר לפי התאמה»; nonce+capability נאכפים (ניסיון POST בלי nonce → לא נשמר).

- **home:** רגרסיה — עדיין נערך ומרונדר זהה (לא נגענו).
