### 2.2 תת-משימה A — ענף QR schema ב-`ea-w2-seo-schema.php`
הוסף בלוק חדש לפני `return $graph;` (L235), במבנה זהה לענפי הקיימים:

1. **זיהוי QR:** `$obj = get_queried_object();` אם `$obj instanceof WP_Post` — חשב
   `$parent_slug = $obj->post_parent ? (string) get_post_field('post_name', $obj->post_parent) : '';`.
   נכנס לענף אם `'qr' === $parent_slug` (ילד QR) **או** `'qr' === $obj->post_name` (ה-hub).
2. **node `Article`:** `@type=Article`, `@id=$site.'#/schema/article/qr/'.$obj->post_name`, `headline=get_the_title($obj)`,
   `url=get_permalink($obj)`, `description` = תיאור גזור (ראה §2.4 QR-branch — אותו מקור), `mainEntityOfPage=get_permalink`,
   `author=array('@id'=>$person_id)`, `publisher=array('@id'=>$biz_id)`. (`$person_id`/`$biz_id` כבר מוגדרים L29-30.)
3. **node `VideoObject` per embedded video:** חלץ מזהי YouTube מ-`$obj->post_content` ב-regex
   `#youtube(?:-nocookie)?\.com/embed/([A-Za-z0-9_-]{6,})#`. לכל מזהה ייחודי, בנה VideoObject **במבנה המדויק של
   node ה-mokesh הקיים** (L220-231):
   `@type=VideoObject`, `@id=$site.'#/schema/video/qr-'.$obj->post_name.'-'.$vid`, `name` (כותרת העמוד + אינדקס אם >1),
   `embedUrl='https://www.youtube-nocookie.com/embed/'.$vid`, `thumbnailUrl='https://i.ytimg.com/vi/'.$vid.'/hqdefault.jpg'`,
   `contentUrl='https://youtu.be/'.$vid`, `about=array('@id'=>$person_id)`, `publisher=array('@id'=>$biz_id)`.
   **`uploadDate`:** להשמיט אם לא ידוע (אין להמציא תאריך — VideoObject תקין בלעדיו, רק לא זכאי לשדה-תאריך ב-rich-result).
4. אם אין וידאו בעמוד — פולט רק את node ה-Article (ללא VideoObject).

### 2.3 תת-משימה B — ענף press/shows-heritage schema
הוסף מפה חדשה + בלוק, במבנה ענף ה-Service (L104-117):
```
$ea_article_pages = array(
    'press'          => array( 'type' => 'CollectionPage', 'name' => 'עיתונות ותקשורת — אייל עמית' ),
    'shows-heritage' => array( 'type' => 'CollectionPage', 'name' => 'מורשת והופעות — אייל עמית' ),

```
אם `is_page()` ו-`isset($ea_article_pages[$slug])` → פולט node: `@type` מהמפה, `@id=$site.'#/schema/'.$type.'/'.$slug`,
`name`, `url=home_url('/'.$slug.'/')`, `description` (מ-`$map` §2.4), `isPartOf=array('@id'=>$site.'#/schema/website')` (מזהה ה-WebSite של Yoast),
`publisher=array('@id'=>$biz_id)`.

### 2.4 תת-משימה C — meta-description ב-`inc/seo-head-fallbacks.php`
1. **press + shows-heritage:** הוסף ל-`$map` (אחרי L50, לפני סגירת המערך L51) שתי כניסות עם copy אמיתי (≤155 תווים):
   - `'press' => 'אייל עמית בתקשורת — כתבות, ראיונות ואזכורים על המרכז לטיפול בנשימה באמצעות דיג׳רידו, שיטת cbDIDG והספרים.'`
   - `'shows-heritage' => 'מורשת והופעות — הופעות, מופעי דיג׳רידו וסיפור המורשת של אייל עמית והמרכז לטיפול בנשימה בפרדס חנה.'`
2. **48 QR (ללא הוספת 48 כניסות):** הוסף ענף לפני ה-lookup הסופי (לפני L72), במבנה fallback ה-excerpt הקיים (L57-62):

3. **ה-hub `/qr/` (כניסה קאנונית — לא נתונה לבחירת הבונה):** הוסף ל-`$map` (יחד עם press/shows-heritage בסעיף 1)
