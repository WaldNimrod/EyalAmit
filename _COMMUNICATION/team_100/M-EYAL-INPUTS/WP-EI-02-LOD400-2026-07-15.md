# WP-EI-02 — LOD400: גלריות ספרים ומוצר — תמונות אמיתיות

## 0. תקציר ההחלפה (swap-on-receipt)

הצוות כבר השתיל **סלוטי גלריה זוהרים** («ממתין לאישור») בעמודי הספרים, ו-**במכוון לא בנה** גלריות מוצר (כדי לא להציג placeholder אפור). הקליטה = החלפת סלוט → תמונה אמיתית, או הוספת סקשן גלריה מוכן למוצרים. אין רכיב חדש: `template-parts/chapters/parts/gallery.php` כבר מטפל גם בפריט אמיתי (`image`+`alt`+`cap`) וגם בסלוט `pending`.

- **מה אייל מספק:** 2–3 תמונות רלוונטיות לכל פריט (Drive / וואטסאפ), **או** אישור להשאיר את המצב הזמני, **או** הוראה לדחות גלריות לשלב מאוחר.
- **מקור/placeholder שמוחלף:** נוצר ב-**WP-S4-02** (סלוטי גלריה + כריכות) ומעוגן במנגנון ה-`pending` ב-`gallery.php`.

## 2. מוסכמת מדיה ושמות קבצים (חובה על team_10)

- יעד: `site/wp-content/themes/ea-eyalamit/assets/images/`.
- ספרים: `assets/images/<book-slug>-NN-<desc>.jpg` — תואם ל-POC של כושי שכבר במאגר (`kushi-01-blantis-1.jpg`, `kushi-02-eyal-italy.jpg`, `kushi-03-screenshot-2013.png`, `kushi-04-sinai.jpg`). דוגמה לוכתבת: `vekatavta-01-<desc>.jpg`.
- מוצרים: `assets/images/shop/<product-slug>-NN.jpg` (תיקיית `shop/` חדשה תחת `assets/images/`).
- כל תמונה: `alt` עברי תיאורי (חובה נגישות), `cap` אופציונלי. פורמט web-ready (JPG/PNG/WebP), רוחב ≤ 2000px.

## 4. יעדי החלפה מדויקים — מוצרים (הוספת סקשן חדש)

ל-5 קבצי ה-defaults: `didgeridoos`, `bags`, `stands-storage`, `stand-floor`, `repair` — **אין** כרגע סקשן גלריה. עם קבלת תמונות, team_10 מוסיף מערך סקשן `gallery` בתוך `'sections'` (מומלץ לפני ה-`product-cta` הסוגר):
```php
array(
	'part' => 'gallery',
	'args' => array(
		'chap'  => 'גלריה',
		'title' => 'תמונות מוצר',
		'items' => array(
			array( 'image' => 'assets/images/shop/<slug>-01.jpg', 'alt' => '<תיאור>', 'cap' => '' ),
			array( 'image' => 'assets/images/shop/<slug>-02.jpg', 'alt' => '<תיאור>', 'cap' => '' ),
		),
	),
),
```
אם אייל דוחה גלריות מוצר — לא מוסיפים סקשן (נשאר המצב המכוון ללא placeholder אפור); מתועד בקליטה.

## 5. קריטריוני קבלה (מדידים)

1. אף עמוד ספר לא מרנדר `gfig--pending` / `.ea-pending-approval` בסקשן הגלריה שהוחלף: `rg -n "'pending'\s*=>\s*true" site/wp-content/themes/ea-eyalamit/inc/chapters/defaults/{vekatavta,tsva-bekahol}-defaults.php` → 0 (לפריטים שהוחלפו).
2. כל `image` שהוזן קיים בפועל: הקובץ נמצא תחת `assets/images/` וה-`<img src>` בחי מחזיר 200 (`qa_probe.mjs` / `curl -I`).
3. לכל תמונה יש `alt` לא-ריק (בדיקת נגישות — grep על ה-defaults + snapshot).
4. עמודי מוצר שקיבלו גלריה: הסקשן מרנדר ותמונות נטענות; עמודים שנדחו — ללא סקשן גלריה, ללא placeholder אפור.
