## 5. RTL→LTR mirroring decisions
**Logical properties (auto-flip, verified PASS):** `padding-inline-start` (list bullets → LEFT), `inset-inline-start` (WhatsApp float → bottom-LEFT), `margin-inline:auto` centering. The deployed W2-08 CSS is genuinely LTR-clean.

**Physical-property RTL bleed found (must be addressed at S3/team_10 — demonstrated correctly in mockup, not silently patched):**
- **M4** `.ea-footer__social` uses `flex-direction:row-reverse; justify-content:flex-end` — physical, does NOT flip with `dir`. Mirror = `row` + `flex-start`.
- **M5** `.ea-footer__nav` uses `justify-content:flex-end` — physical. Mirror = `flex-start`.
- **M6** `.ea-skiplink` uses physical `right:0`. Mirror = `inset-inline-start:0`.
- **M10** Composition decision: nav language control on /en = `עברית` pill (`hreflang="he" lang="he"`), reciprocating the B03 hreflang already emitted by `ea_w2_08_hreflang()`.

Full checklist M1–M10 in `narrative/composition-notes.md`.
