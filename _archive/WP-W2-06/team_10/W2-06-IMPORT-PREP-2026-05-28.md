## 7. Import Instructions Summary

```bash
# From the WordPress installation directory:

# 1. Install importer plugin (if not already active)
wp plugin install wordpress-importer --activate

# 2. Run the import script (from repo root)
bash scripts/blog-import.sh --wp-path /path/to/wordpress

# 3. Flush permalinks after import
wp rewrite flush --hard
```

**Author mapping:** The WXR maps all posts to `admin` (user ID 1). The `--authors=mapping` flag is used; if the admin login matches user ID 1 in the target site, no interactive prompt is needed.

**Attachments:** `--skip=image_resize` is set. Attachment URLs point to the old domain — linked images will need to be re-uploaded separately or served from the legacy CDN if still accessible.
